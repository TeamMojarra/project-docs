## Descripción
Reservent es una plataforma web pensada para negocios pequeños que necesitan organizar reservas y accesos de manera simple y ordenada. Desde el sistema se pueden publicar servicios o actividades con horarios o cupos disponibles, y los clientes pueden confirmar su reserva en pocos pasos. Al finalizar, el sistema envía una confirmación por correo con un código QR único, que luego se utiliza en el check-in para validar la asistencia y evitar duplicados o fraudes.
